####  V1.0 First Release

## How to run...? 

#### 01. Download [Setup.sh](https://github.com/Infamousmick/infaScript/blob/master/setup.sh)
#### 02. Copy and paste in the terminal and press ENTER:
```
chmod u+x setup.sh
./setup.sh
```
     
#### 02. Run InfaScript.sh and ENJOY :D
```
cd infaScript
./Infascript.sh
```
