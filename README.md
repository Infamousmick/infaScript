## How to start ?

### 01. Start Shizuku or Grant root access to your Termux.
### 02. Type this in Termux and press enter :
```
infa -s
```

## Infa usage:

### Update InfaScript:
```
infa -u
```

### Remove InfaScript:
```
infa -r
```

### Start InfaScript:
```
infa -s
```
